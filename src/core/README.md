# Core Layer (Business Logic)

Сердце приложения. Здесь живут банковские правила, сценарии использования (Use Cases) и управление транзакциями.

## Здесь
- Реализация бизнес-сценариев: `create_account`, `transfer_funds`, `calculate_interest`.
- Валидация бизнес-правил (достаточно ли средств? активен ли счет?).
- Управление атомарностью операций (начало/коммит/откат транзакций БД).
- Генерация событий домена (если используется событийная модель).
