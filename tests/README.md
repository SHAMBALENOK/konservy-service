# Tests

Набор тестов для обеспечения качества и безопасности банковского ПО.

## 📂 Структура
- `unit/`: Тесты бизнес-логики (`core/`) без БД и сети. Быстрые, изолированные.
- `integration/`: Тесты взаимодействия с БД, внешними API и полными HTTP-циклами.
- `conftest.py`: Общие фикстуры (test DB session, mock clients, auth tokens).
